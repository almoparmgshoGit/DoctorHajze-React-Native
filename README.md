# DoctorApp - نظام إدارة حجوزات وطوابير العيادة

تطبيق هاتف محمول مخصص لطبيب واحد لإدارة المواعيد وتتبع طابور الانتظار (Live Turn Tracking) في مختلف فروع العيادة، مبني باستخدام **React Native** و **Expo**.

---

## 🚀 المميزات الحالية (Current Features)

- **نظام تسجيل دخول وانشاء حساب**: دعم كامل للمرضى عبر البريد الإلكتروني وكلمة المرور.
- **واجهة مستخدم عصرية**: تصميم أنيق يعتمد على **NativeWind (Tailwind CSS)** مع واجهة سهلة التنقل.
- **التنقل (Navigation)**: نظام تنقل احترافي يشمل القوائم الأساسية (الرئيسية، حجوزاتي، الإعدادات).
- **مكونات مخصصة**: Header ديناميكي مع إشعارات وعناوين متغيرة حسب الصفحة.
- **تكامل مع Firebase**: استخدام Firebase للمصادقة وتخزين البيانات اللحظية (Real-time).

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **Framework**: Expo / React Native
- **Styling**: NativeWind (Tailwind CSS)
- **Navigation**: React Navigation (Bottom Tabs & Native Stack)
- **Backend**: Firebase (Authentication & Firestore)
- **Icons**: Lucide React & Ionicons

---

## 📅 خارطة الطريق للتطوير المستقبلي (Development Roadmap)

نركز في هذا التطبيق على تحويل تجربة الانتظار في العيادة إلى تجربة رقمية منظمة:

### 1️⃣ المرحلة الأولى: نظام الحجز والتتبع اللحظي
- [ ] **تتبع الدور (Live Turn Tracking)**: إمكانية رؤية الرقم الحالي في الطابور والوقت المتوقع للدخول.
- [ ] **إدارة المواقع**: اختيار فرع العيادة (في حال وجود أكثر من مقر للطبيب).
- [ ] **حجوزاتي السابقة**: صفحة مخصصة لعرض تاريخ الزيارات السابقة والتقارير المرتبطة بها.

### 2️⃣ المرحلة الثانية: الملف الشخصي والإعدادات
- [ ] **الملف الشخصي (Profile)**: إدارة البيانات الشخصية للمريض (الاسم، العمر، التاريخ الطبي المصغر).
- [ ] **صفحة الإعدادات**: التحكم في التنبيهات، تغيير كلمة المرور، وتخصيص تجربة التطبيق.
- [ ] **نظام التنبيهات**: إرسال تنبيه للمريض عندما يقترب دوره (مثلاً: "باقي 3 حالات قبل دورك").

### 3️⃣ المرحلة الثالثة: الميزات المتقدمة
- [ ] **إدارة الملف الطبي**: تمكن المريض من رفع صور التحاليل والأشعة السابقة لتكون متاحة للطبيب أثناء الكشف.
- [ ] **دعم اللغات**: توفير واجهة كاملة باللغتين العربية والإنجليزية.

## 💻 طريقة التشغيل (Setup Guide)

1. قم بتحميل المستودع:
   ```bash
   git clone https://github.com/almoparmgshoGit/DoctorApp.git
   ```
2. قم بتثبيت المكتبات:
   ```bash
   npm install
   ```
3. تشغيل المشروع:
   ```bash
   npx expo start
   ```
